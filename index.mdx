title: "zoxide — A smarter cd command"
description: "Jump to your most-used directories in just a few keystrokes. zoxide learns your habits and works with all major shells."
---

zoxide is a smarter `cd` command, inspired by z and autojump. It remembers which directories you use most frequently, so you can jump to them with just a few keystrokes — on any major shell.

```bash
z foo          # cd into highest ranked directory matching foo
z foo bar      # cd into highest ranked directory matching foo and bar
zi foo         # cd with interactive selection using fzf
```

## Get up and running

<Steps>
  <Step title="Install the zoxide binary">
    Install zoxide via your system package manager, `cargo`, or the install script:

    ```bash
    curl -sSfL https://raw.githubusercontent.com/ajeetdsouza/zoxide/main/install.sh | sh
    ```
  </Step>
  <Step title="Add zoxide to your shell">
    Add the init command to your shell config file. For example, in `~/.bashrc`:

    ```bash
    eval "$(zoxide init bash)"
    ```

    See [shell setup](/shell-setup) for instructions for every supported shell.
  </Step>
  <Step title="Start jumping">
    Open a new terminal session and start navigating. zoxide learns as you go:

    ```bash
    cd ~/projects/my-app    # visit some directories first
    cd ~/documents/notes

    z my-app                # jump back to ~/projects/my-app
    z notes                 # jump back to ~/documents/notes
    ```
  </Step>
</Steps>

## Why zoxide?

<CardGroup cols={2}>
  <Card title="Frecency ranking" icon="chart-line">
    Combines frequency and recency to always surface the most relevant directory for your query.
  </Card>
  <Card title="All major shells" icon="code">
    Native support for Bash, Zsh, Fish, PowerShell, Nushell, Elvish, Xonsh, Tcsh, and any POSIX shell.
  </Card>
  <Card title="Interactive selection" icon="magnifying-glass">
    Use `zi` to fuzzy-search through matching directories interactively with fzf.
  </Card>
  <Card title="Cross-platform" icon="globe">
    Works on Linux, macOS, Windows (including WSL, Git Bash, Cygwin), BSD, and Android.
  </Card>
</CardGroup>
