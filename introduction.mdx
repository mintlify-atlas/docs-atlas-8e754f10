title: "Introduction"
description: "zoxide is a smarter cd command that learns which directories you use most, so you can jump to them in just a few keystrokes."
---

zoxide is a smarter `cd` command, inspired by z and autojump. It tracks which directories you visit and ranks them by how frequently and how recently you use them — so the right directory is always one short query away.

```sh
z foo              # cd into highest ranked directory matching foo
z foo bar          # cd into highest ranked directory matching foo and bar
z foo /            # cd into a subdirectory starting with foo

z ~/foo            # z also works like a regular cd command
z foo/             # cd into relative path
z ..               # cd one level up
z -                # cd into previous directory

zi foo             # cd with interactive selection (using fzf)

z foo<SPACE><TAB>  # show interactive completions (bash 4.4+ / fish / zsh only)
```

## How it compares

| Feature | `cd` | z / autojump | zoxide |
| --- | --- | --- | --- |
| Jump by partial name | No | Yes | Yes |
| Multi-keyword matching | No | No | Yes |
| Interactive fuzzy selection | No | No | Yes (`zi`) |
| All major shells | Yes | Partial | Yes |
| Written in Rust (fast) | — | No | Yes |
| Import from z / autojump | — | — | Yes |

Unlike plain `cd`, zoxide remembers where you go and ranks directories by a frecency score — a combination of how **frequently** and how **recently** you visited them. The more you use a directory, the higher it ranks. Entries that haven't been visited in a long time gradually age out.

Unlike z and autojump, zoxide supports multi-keyword queries (`z foo bar` matches a directory containing both "foo" and "bar"), ships with native interactive selection via `zi`, and works consistently across all major shells without relying on shell plugin managers.
