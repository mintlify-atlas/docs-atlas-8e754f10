---
title: "Third-party integrations"
description: "Plugins and native support for editors, file managers, launchers, terminal multiplexers, and more."
---

zoxide has a rich ecosystem of third-party integrations. Many applications have added native support — meaning no extra plugin is required — while others are supported through community-maintained plugins.

<AccordionGroup>
  <Accordion title="Text editors & IDEs">
    | Application | Description | Plugin |
    | ----------- | ----------- | ------ |
    | [Emacs](https://www.gnu.org/software/emacs/) | Text editor | [zoxide.el](https://gitlab.com/Vonfry/zoxide.el) |
    | [Vim](https://github.com/vim/vim) / [Neovim](https://github.com/neovim/neovim) | Text editor | [zoxide.vim](https://github.com/nanotee/zoxide.vim) |
    | [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) | Fuzzy finder for Neovim | [telescope-zoxide](https://github.com/jvgrootveld/telescope-zoxide) |
  </Accordion>

  <Accordion title="File managers">
    | Application | Description | Plugin |
    | ----------- | ----------- | ------ |

    | [joshuto](https://github.com/kamiyaa/joshuto) | File manager | Natively supported |
    | [lf](https://github.com/gokcehan/lf) | File manager | See the [wiki](https://github.com/gokcehan/lf/wiki/Integrations#zoxide) |
    | [nnn](https://github.com/jarun/nnn) | File manager | [nnn-autojump](https://github.com/jarun/nnn/blob/master/plugins/autojump) |
    | [ranger](https://github.com/ranger/ranger) | File manager | [ranger-zoxide](https://github.com/jchook/ranger-zoxide) |

    | [xplr](https://github.com/sayanarijit/xplr) | File manager | [zoxide.xplr](https://github.com/sayanarijit/zoxide.xplr) |
    | [yazi](https://github.com/sxyazi/yazi) | File manager | Natively supported |

    <Note>
      "Natively supported" means the application ships built-in zoxide
      integration — no extra plugin installation is needed.
    </Note>
